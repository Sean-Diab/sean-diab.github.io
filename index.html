Projects

Reviser

Solo research project · Dec 2025–Apr 2026

Revision-capable language modeling via autoregressive cursor actions over a mutable token canvas ( Insert/Move/Stop ).

  • Built model, data pipeline, training loop, inference tooling, and evaluation stack in PyTorch.
  • Ran 3k-sample pairwise LLM-judge arenas; achieved 61.3% (100M) and 54.4% (300M) 1v1 win rate vs matched AR baselines, and 85.9%/78.3% vs SEDD/MDLM, on response quality.
  • Released code and paper; filed U.S. Provisional Patent Application No. 63/977,897.
